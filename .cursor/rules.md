# VolunPeers - Cursor Rules

## Project Overview
VolunPeers is a Ruby on Rails 7.1 volunteering platform with chat, events, and user management.

## Tech Stack
- Rails 7.1.3, Ruby 3.3.9
- PostgreSQL, Redis (ActionCable)
- Devise (auth), Pundit/Cancancan (authorization)
- Bootstrap 5, Slim templates, Sass
- Hotwire (Turbo + Stimulus)
- Cloudinary (images), Geocoder (locations)
- RSpec (testing)

## Conventions
- Use `bundle exec` for Rails commands
- Follow Rails conventions for RESTful routes
- Use Pundit policies for authorization
- Write tests with RSpec
- Format Slim templates with proper indentation
- Use Stimulus controllers for interactivity
- Use Turbo Streams for AJAX updates
